Lists

We can create several types of lists within a document. The first type of list we will discuss is the unordered list. As the name implies, this type of list simply itemizes several items withouth numbering them or ordering them.

  • One
  • Two
  • Three

Conversely, the ordered list does number a list of items.

  1. One
  2. Two
  3. Three
